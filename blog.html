1) What Is DOM And What Is The Purpose Of DOM?

DOM means Document Object Model (DOM), which is a programming interface for HyperText Markup Language (HTML) and Extensible markup language (XML) documents. It defines the logical connection of documents and the way by which a document is accessed and manipulated.

Purpose of DOM

  • To create Dynamic Web Page.
  • Main purpose of DOM is to respond to the user actions (like clicks, inputs, or scrolls).
  • To update content

    • 3) What Is Event Bubble?

    Event bubbling is a process that starts with the element that triggered the event and then bubbles up to the containing elements in the hierarchy. In event bubbling, the event is first captured and handled by the innermost element and then propagated to outer elements.

    4) What Is A Callback Function And Why Will You Use It?

    A callback function in javascript is a function which is to be executed after another function has finished its execution.

    We use callback functions to make sure that a function is only going to get executed after a specific task is completed and not before its completion.
